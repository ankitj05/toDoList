# Express based Simple toDo List App
Simple toDo list app, using Express, MongoDB and basic css.

## Development Setup
### Requirements:
- MongoDB
- node (v8, v12, v14) + npm

### Setup:
```shell
npm install
npm start
```

# Note
Please create a .env file in the root directory and connect to your mongoDB in that.
